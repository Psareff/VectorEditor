# VectorEditor by Psareff, September 2022
This vector editor consists of vector canvas, shapes such as circles, triangles

#------------------------------------------------------------------------------

Shapes have these attributes:
- Fill color
- Contour color

Triangle:
- Three vertices
- Three sides
- Three side lengths

Circle:
- Radius
- Center coordinates

Canvas have these attributes:
- Size on X axis
- Size on Y axis
